# _Structure (текущая версия инструкций по структуре)

Это **текущая** версия инструкций по структуре репозитория и компании — единый источник правды для агента.

- **Версии и отчёты** по разработке структуры хранятся в **`_Structure-Management/`** (история версий v0.1 … v0.4, Reports).
- При **релизе новой версии** (например v0.5) содержимое соответствующей папки в `_Structure-Management/` копируется в `_Structure/`, заменяя текущие файлы.

Инструкции для агента по работе с обновлениями версий структуры — в [AGENT_Structure_Version_Workflow.md](./AGENT_Structure_Version_Workflow.md).
