# 01-Management (Управление)

**Зачем:** Зона для топ-менеджмента и принятия стратегических решений.

**Что хранить:** Протоколы совещаний директоров, стратегические планы развития, отчетность перед инвесторами/учредителями. Это 'мозг' компании.

### Структура подразделов:
*   **01-01-Strategy**: Стратегические планы 1/3/5 лет.
*   **01-02-Board-Meetings**: Протоколы совета директоров.
*   **01-03-Reports**: Сводная отчетность для акционеров.
*   **01-04-Partnerships**: Ключевые партнерства и альянсы.

Управление структурой: текущие инструкции для агента — **`_Structure/`**; версии и отчёты по разработке структуры — **`_Structure-Management/`** (в корне).
