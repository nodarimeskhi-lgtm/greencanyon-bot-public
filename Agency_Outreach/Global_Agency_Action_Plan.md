## 2. Tailored Presentation Categories / მორგებული პრეზენტაციების ტიპები

One size does NOT fit all. We need 3 distinct decks:
ერთი პრეზენტაცია ყველას არ მოერგება. გვჭირდება 3 განსხვავებული ვერსია:

### Type 1: The "Family Retreat" Deck (For Saudi/Kuwait End-Users)
*   **Focus**: **Privacy (100%)**, Cool Weather (20°C), Halal Food, Large Villas (4+ BR).
*   **Keywords**: "Sanctuary", "Tradition", "Safe", "No Humidity".
*   **Visuals**: High walls, foggy canyon, family dinners, no alcohol visible.

### Type 2: The "Yield Hunter" Deck (For Dubai/Israel Investors)
*   **Focus**: **ROI (12%+)**, Management Company, Capital Appreciation, Exit Strategy.
*   **Keywords**: "Passive Income", "High Occupancy", "Dollar Asset", "Eagle Hills effect".
*   **Visuals**: Charts, Numbers, Construction timeline, Booking.com screenshots of competitors.

### Type 3: The "Eco-Explorer" Deck (For Europe/Expats)
*   **Focus**: **Sustainability**, Adventure, Hiking, "Green" Architecture.
*   **Keywords**: "Carbon Footprint", "Nature Connection", "Adventure", "Glass & Wood".
*   **Visuals**: Hiking trails, waterfalls, yoga decks, starry nights.
