# Руководство пользователя: Гибридная система "Человек-ИИ"

## Введение
Это руководство объясняет, как работать с файловой структурой Galt, которая теперь адаптирована для совместной работы людей и Искусственного Интеллекта (ИИ).
Наша цель: Сделать так, чтобы ИИ понимал наши правила и контекст, не путался в черновиках и помогал нам быстрее.

---

## 1. Главные принципы

### 🧪 Песочница (Drafts) — для экспериментов
В каждой рабочей папке может быть подпапка `_Drafts`.
*   **Что это:** Место для черновиков, идей, споров с ИИ.
*   **Правило:** ИИ считает всё, что здесь лежит, *несуществующим* для официальной работы компании. Это черновики.
*   **Пример:** Вы просите ИИ "Напиши 5 вариантов стратегии". ИИ положит их сюда.

### 📄 Официальные документы — в корне папки
Файлы, лежащие прямо в рабочих папках (например, `01-Management/Vacation_Policy.md`), считаются **истиной**.
*   **Правило:** Чтобы файл стал "официальным", он должен иметь специальную "шапку" (метаданные).

---

## 2. Ваш новый рабочий процесс

### Шаг 1: Создание (Brainstorming)
Вы говорите ИИ: *"Предложи структуру отдела продаж"*.
ИИ создает файл в `01-Management/_Drafts/Sales_Structure_v1.md`.
Вы обсуждаете, меняете, критикуете. Система "не видит" этот файл как правило.

### Шаг 2: Утверждение (Promotion)
Вы решили: "Да, этот вариант отличный".
1.  Переместите файл из `_Drafts` в основную папку (например, `01-Management/`).
2.  **ВАЖНО:** Добавьте в начало файла блок метаданных (YAML Frontmatter):

```yaml
---
id: structure-sales-v1
status: approved
owner: Head of Sales
ai_access: true
---
```

Только после этого ИИ-агенты (например, при поиске "Какая у нас структура?") начнут учитывать этот файл.

---

## 3. Скрытые помощники

Вы можете заметить файл `_AI_CONTEXT.md` в папках.
*   **Не удаляйте его.**
*   В нем записаны инструкции для ИИ именно по этой папке (например: "В этой папке хранятся только договоры, naming convention: `Contract_Name_Date`").
*   Если хотите изменить правила папки для ИИ — редактируйте этот файл.

## Краткая памятка

| Действие | Где делать? | Нужны метаданные? | Видит ли глобальный ИИ? |
| :--- | :--- | :--- | :--- |
| **Эксперимент / Черновик** | `_Drafts/` | Нет | ❌ Нет |
| **Готовый документ** | Корневая папка | ✅ **ДА** | ✅ Да |
