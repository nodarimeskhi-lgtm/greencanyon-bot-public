# _Structure-Management (Управление структурой)

Мета-раздел: правила, регламенты и теоретические обоснования структуры базы знаний и компании. Расположен в корне репозитория (вне нумерованной структуры 00–99).

**Текущая версия инструкций для агента** вынесена в корень: **`_Structure/`**. Там хранится один источник правды (Structure_Instruction, Rules_Instruction, JD и т.д.). Здесь в `_Structure-Management/` — только версии (v0.1 … v0.4) и отчёты (Reports/). При утверждении новой версии её содержимое копируют в `_Structure/`.

## Содержание
- [v0.1](./v0.1/) — Первая версия структуры (детальная иерархия, теоретическое обоснование, анализ).
- [Reports](./Reports/) — Отчёты по работе над структурой (аудит, планы, changelog).
