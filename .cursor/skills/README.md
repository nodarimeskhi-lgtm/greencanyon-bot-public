# Galt Role-Based Skills

ИИ-скиллы для поддержки ключевых ролей строительной компании Galt.

## Концепция

Компания Galt строит **гибридную систему управления**, где человеческая экспертиза дополняется возможностями ИИ-агентов. Каждый скилл соответствует должности из [Organizational_Roles.md](../01-Management/01-05-Structure-Management/v0.3/Organizational_Roles.md).

## Принципы гибридной работы

### Делегируется ИИ
- Сбор и анализ данных
- Мониторинг и алерты
- Генерация черновиков документов
- Рутинные расчёты
- Поиск информации
- Автоматизация процессов

### Остаётся за человеком
- Принятие решений
- Переговоры и отношения
- Финальное утверждение
- Нестандартные ситуации
- Этические вопросы
- Лидерство и мотивация

## Как использовать

### Активация скилла
Скиллы активируются автоматически по ключевым словам в запросе или по контексту работы.

**Примеры триггеров:**
- "Подготовь отчёт для Совета" → `galt-ceo`
- "Нужна вакансия инженера" → `galt-hr`
- "Проанализируй бюджет проекта" → `galt-cfo`
- "Проверь договор подряда" → `galt-legal`
- "Статус проекта Tsalka" → `galt-pm`

### Явное указание роли
Можно явно указать роль:
- "Работаю как CFO, проанализируй cash flow"
- "Для PM: подготовь еженедельный статус"

## Структура скилла

Каждый скилл содержит:
1. **Контекст роли** — описание должности и подчинения
2. **Ключевые задачи** — что агент может делать
3. **Доступ к ресурсам** — какие папки релевантны
4. **Форматы вывода** — шаблоны документов
5. **Ограничения** — что НЕ делегировать ИИ
6. **Взаимодействие** — связь с другими ролями
