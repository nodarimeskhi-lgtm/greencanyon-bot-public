# ბაზრის ანალიზი და გასაყიდი ფასის სტრატეგია - Green Canyon Studio

თქვენი მოთხოვნის საფუძველზე, გავაანალიზე 45მ² სტუდიოს გასაყიდი ღირებულება, რომლის სავარაუდო თვითღირებულება (CAPEX) შეადგენს **$72,500**-ს (ინფრასტრუქტურის, რემონტის და ავეჯის ჩათვლით).

## 1. ფასწარმოქმნის მოდელი (Cost-Plus Method)

დეველოპერული პროექტებისთვის სტანდარტული მოგების მარჟა მერყეობს 25%-დან 40%-მდე, თუმცა პრემიუმ/ნიშურ პროექტებში (როგორიცაა "Eco Village" და "Halal" კონცეფცია) მარჟა შეიძლება იყოს უფრო მაღალიც.

| კომპონენტი | ღირებულება | შენიშვნა |
| :--- | :---: | :--- |
| **თვითღირებულება (CAPEX)** | **$72,500** | ~$1,610 / მ² |
| **სამიზნე მარჟა (Margin)** | | |
| - მინიმალური (25%) | + $18,125 | სტანდარტული დეველოპერული მოგება |
| - საშუალო (35%) | + $25,375 | პრემიუმ სეგმენტი |
| - მაღალი (50%) | + $36,250 | ექსკლუზიური / Luxury |
| **სარეკლამო/გაყიდვის ხარჯი** | ~$5,000 | ბროკერის საკომისიო და მარკეტინგი |

### სარეკომენდაციო გასაყიდი ფასი:

*   **მინიმალური ზღვარი:** $72,500 + $18,125 + $5,000 = **~$95,500** ($2,120 / მ²)
*   **ოპტიმალური ფასი:** $72,500 + $25,375 + $5,000 = **~$103,000** ($2,290 / მ²)
*   **პრემიუმ ფასი:** $72,500 + $36,250 + $5,000 = **~$114,000** ($2,530 / მ²)

## 2. ბაზრის შედარებითი ანალიზი (Benchmarking)

არის თუ არა ეს ფასი ($2,100 - $2,500 / მ²) რეალური საქართველოს ბაზრისთვის? შევიდაროთ ანალოგებს:

*   **თბილისი (ვაკე/მთაწმინდა - პრემიუმ):** დასრულებული, გარემონტებული აპარტამენტები იყიდება **$2,500 - $3,500 / მ²** ფარგლებში.
    *   *Green Canyon-ის უპირატესობა:* ეკოლოგია, პრივატულობა, უნიკალური ხედები.
*   **გუდაური / ბაკურიანი (პრემიუმ):** მაღალი კლასის აპარტამენტები (მაგ. Axis, Redco) იყიდება **$1,800 - $2,500 / მ²**.
    *   *Green Canyon-ის უპირატესობა:* წალკა ახალი, აუთვისებელი და უფრო ექსკლუზიური მიმართულებაა (ნაკლები სიმჭიდროვე).
*   **საირმე / აბასთუმანი:** ახალი კურორტების ფასები მერყეობს **$1,500 - $2,200 / მ²**-ს შორის.

## 3. დასკვნა და რეკომენდაცია

დიახ, **$95,000 - $105,000** დიაპაზონი (დაახლოებით $2,100 - $2,300 კვ.მ-ზე) აბსოლუტურად რეალურია და შეესაბამება საქართველოს პრემიუმ უძრავი ქონების ბაზარს, განსაკუთრებით შემდეგი ფაქტორების გათვალისწინებით:

1.  **უნიკალური კონცეფცია:** "ჰალალ" სერტიფიცირებული, პრივატული ეკო-ვილაბი - კონკურენტი პრაქტიკულად არ ჰყავს.
2.  **დასრულებული პროდუქტი:** მყიდველი იღებს არა "თეთრ კარკასს", არამედ სრულად გარემონტებულ, ტექნიკით აღჭურვილ და ჭკვიანი სისტემებით მართვად სახლს (Turnkey).
3.  **GCC აუდიტორია:** სამიზნე სპარსეთის ყურის ქვეყნების მყიდველისთვის ეს ფასი ($100k) სრულიად მისაღებია "მეორე სახლისთვის" (Second Home) გრილ კლიმატში.

**რეკომენდაცია:**
გაყიდვების დაწყება რეკომენდებულია **$98,000**-დან (Pre-sale ფასი), ხოლო მშენებლობის დასრულებისას ფასმა შეიძლება მიაღწიოს **$120,000**-ს.
