# Оценка: AI Skills для архитектурной графики vs Специализированные инструменты

## Контекст теста

Протестирован `canvas-design` skill в Cursor IDE для генерации архитектурного постера мастерплана проекта Tsalka (Alpine Village, 170 коттеджей, эко-ферма, Грузия).

---

## 1. Результаты теста canvas-design skill

### Что получилось хорошо
- **Картографический стиль**: Топографические контурные линии, масштабная линейка, компас -- профессиональный вид
- **Информационная графика**: Легенда, данные проекта, координаты, цитата -- все структурировано
- **Цветовая палитра**: Сдержанные земляные тона, согласованные с философией "Alpine Cartography"
- **Типографика**: Множество шрифтов из библиотеки canvas-fonts, иерархия заголовков
- **Выходной формат**: 2400x3200px при 300 DPI (203x271mm) -- готово к печати

### Ограничения
- **Абстрактность**: Это скорее арт-постер, чем точный мастерплан -- нет реальной привязки к местности
- **2D Pillow**: Генерация через PIL -- плоские фигуры, нет перспективы, нет фотореализма
- **Нет рельефа**: Контурные линии декоративные, не основаны на реальных данных высот
- **Нет текстур**: Отсутствуют текстуры дерева, камня, травы -- только заливки цветом

### Оценка: 6/10 для архитектурной презентации, 8/10 для концептуального постера

## 3. Рекомендации по инструментам для Tsalka

### Стадия концепции (текущая)
**Рекомендуется**: Canvas-Design + Midjourney v7

- Canvas-Design: диаграммы, инфографика, картографические постеры
- Midjourney: атмосферные рендеры для вдохновения и презентации инвесторам
- Three.js skills: интерактивные 3D-превью для веб-сайта проекта

### Стадия проектирования (P)
**Рекомендуется**: PromeAI / Architect AI + Three.js

- PromeAI: быстрая конвертация скетчей в рендеры (< 10 сек)
- Architect AI: 20+ архитектурных стилей, контроль погоды
- Three.js (через установленные skills): интерактивный masterplan viewer на сайте

### Стадия рабочей документации (RD)
**Рекомендуется**: D5 Render / Enscape + BIM

- D5 Render: реальное время, прямая интеграция со SketchUp
- Enscape: если используется Revit/SketchUp -- прямая интеграция
- Twinmotion: бесплатен для малых проектов, ray-tracing

### Маркетинг и продажи
**Рекомендуется**: ArchRender + Canvas-Design

- ArchRender: фотореалистичные рендеры из 3D моделей, неограниченные ракурсы
- Canvas-Design: брошюры, постеры, маркетинговые материалы

## 5. Итоговая оценка

### AI IDE Skills (Cursor / Claude Code / Antigravity)
- **Сильные стороны**: Программный контроль, воспроизводимость, скорость, бесплатность, интеграция в workflow разработки
- **Слабые стороны**: Нет фотореализма, ограничены 2D/программной графикой, требуют навыков кодирования
- **Вердикт**: Идеальны для концептуальной стадии и интерактивных веб-презентаций

### Специализированные AI-рендеринг инструменты (Midjourney, ArchRender, PromeAI)
- **Сильные стороны**: Фотореализм, быстрая генерация, множество стилей
- **Слабые стороны**: Меньше контроля, стоимость подписки, не всегда точны
- **Вердикт**: Лучший выбор для презентаций инвесторам и маркетинга

### BIM-интегрированные рендереры (D5, Enscape, Twinmotion)
- **Сильные стороны**: Максимальная точность, реальное время, прямая связь с проектной моделью
- **Слабые стороны**: Требуют полную 3D модель, высокая стоимость лицензий
- **Вердикт**: Необходимы на стадии рабочего проектирования и строительства

### Общий вывод
Для проекта Tsalka оптимальна **гибридная стратегия**: AI IDE skills для быстрых итераций и веб-интерфейсов + специализированные инструменты для фотореалистичных визуализаций и BIM-рендеринга. Все установленные skills готовы к использованию.
