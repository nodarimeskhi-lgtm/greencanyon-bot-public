# Контекст: 99-Archive

## Назначение
Архив: исторические записи, устаревшие документы. Специальный индекс вне системы Johnny.Decimal.

## Владение (Context Owners)
- **Role**: CEO / Администратор
- **Team**: Администрация
- **Категория по Минцбергу**: Инфраструктура (вне классификации)

## Координация
- **Связанные папки**: Все департаменты (источник архивных документов)
- **Права принятия решений**: Перемещение в архив — владелец документа; удаление — CEO

## Правила для ИИ
- Режим: **Только чтение**. Не создавать новый контент.
- Используется только при перемещении устаревших файлов из других папок.
